# Tilt Ruler - Flutter Sensor App

Aplikasi Flutter sederhana yang memanfaatkan 3 sensor smartphone untuk mengukur kemiringan dan arah kompas secara real-time.

## Sensor yang Digunakan

- **Accelerometer** — Mengukur kemiringan perangkat (pitch & roll)
- **Gyroscope** — Mengukur kecepatan rotasi perangkat
- **Magnetometer** — Menentukan arah mata angin (kompas)

## Fitur

- **Bubble Level** — Waterpass digital visual
- **Tilt Angle** — Sudut kemiringan & rotasi dalam derajat
- **Kompas** — Jarum kompas real-time dengan 8 arah mata angin

## Tech Stack

- Flutter
- Dart
- Package: `sensors_plus`, `flutter_compass`, `provider`

## Cara Menjalankan

```bash
flutter pub get
flutter run
```
